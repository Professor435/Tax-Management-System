# Tax-Management-System
I've created a beautiful, professional Tax Management System with a clean light theme and fully functional dark mode toggle. Here's what's included:
🎨 Design Features
Light Mode (Default)
Clean white and light gray backgrounds
Subtle shadows and borders for depth
Blue-purple gradient accents
High contrast for readability
Professional, modern appearance
Dark Mode
Deep slate backgrounds ( #0f172a to #1e293b )
Glassmorphism effects with transparency
Neon accent colors (blue, purple, emerald)
Reduced eye strain for night use
Smooth transitions between themes
🌓 Dark Mode Implementation
Persistent theme storage (saves to localStorage)
Smooth toggle animation with sun/moon icons
Automatic chart theme switching (updates colors instantly)
CSS class-based switching using Tailwind's dark: variants
All components adapt - tables, cards, inputs, modals
⚡ Key Features
Dashboard
Revenue statistics with trend indicators
Interactive charts (revenue trends, tax distribution)
Recent returns table
Quick action buttons
Taxpayer Management
Register individuals, businesses, corporations
Filter by type (All/Individual/Business/Corporate)
Search functionality
Taxpayer cards with return history
Tax Returns
File different return types
Real-time tax calculation
Status tracking (Draft → Filed → Pending → Paid)
Payment integration
Smart Tax Calculator
Live calculation as you type
Tax bracket breakdown with visual bars
Filing status options (Single/Married)
Deductions and credits support
Optimization tips
Payment Center
Record payments with multiple methods
Payment statistics (Collected/Pending/Overdue)
Transaction history
Export functionality
Analytics & Audit
Collection trend charts
Taxpayer growth visualization
Regional compliance heatmap
Complete audit trail with timestamps
🔧 Technical Highlights
Fully responsive - works on mobile, tablet, desktop
No external frameworks - pure HTML/CSS/JS
Chart.js integration with theme-aware colors
Form validation and error handling
Toast notifications for user feedback
Modal dialogs for all CRUD operations
Smooth animations throughout
🚀 How to Use
Toggle Dark Mode: Click the toggle switch in the sidebar (saves preference)
Navigate: Use sidebar menu or click sections
Add Taxpayer: Click "Register Taxpayer" button
File Return: Select taxpayer and enter income details
Calculate Tax: Go to Tax Calculator, enter income
Record Payment: Select return and process payment
